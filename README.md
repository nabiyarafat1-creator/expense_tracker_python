Expense Tracker

Overview

Expense Tracker is a simple command-line application developed using Python to help users record and monitor their daily expenses. The project was created as a learning exercise to strengthen programming fundamentals and gain practical experience in building software applications.

Features

- Add new expenses with amount details
- View all recorded expenses
- Calculate total spending
- Simple and user-friendly command-line interface

Technologies Used

- Python

Project Structure

expense_tracker.py – Main application file containing the program logic.

README.md – Project documentation.

Learning Outcomes

Through this project, I gained hands-on experience with:

- Python programming fundamentals
- Conditional statements and loops
- Lists and data storage
- User input handling
- Basic problem-solving and application design

Future Improvements

Potential enhancements for future versions include:

- Data storage using files or databases
- Expense categorization
- Monthly spending reports
- Graphical user interface (GUI)
- Data visualization and analytics
